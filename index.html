Persona Labs: Product Recommendation API

Welcome! This guide shows how to:

  1. Generate an API key
  2. Make queries (POST /query)
  3. Send arbitrary payloads (POST /send)
Requests must be authenticated with X-API-Key—so step 1 first.

1 · Generate an API key (POST /generate_api_key)

Request body

{
  "username": "acme-inc"
}

Response

{
  "username": "acme-inc",
  "api_key": "S2evAhxSlfq4P0xegU8VKL9X-fZ5AgK8rfK2mPAD"
}
curl -X POST https://jgypz1bf15gb8c.proxy.runpod.net/generate_api_key \
  -H "Content-Type: application/json" \
  -d '{"username": "acme-inc"}'
import requests, json

resp = requests.post(
    "https://jgypz1bf15gb8c.proxy.runpod.net/generate_api_key",
    json={"username": "acme-inc"},
)
print(resp.json())
const res = await fetch("https://jgypz1bf15gb8c.proxy.runpod.net/generate_api_key", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ username: "acme-inc" })
});
console.log(await res.json());

2 · Retrieve recommendations (POST /query)

Headers

  • Content-Type: application/json
  • X-API-Key: <your-key>

Request body

{
  "basket_query": "blue trucker hat\nblack snapback\nsummer sale"
}

Response

{
  "ids": ["P-20034", "P-15502", "P-00912", "..."],
  "basket_query": "blue trucker hat\nblack snapback\nsummer sale",
  "gpu_used": true
}
curl -X POST https://jgypz1bf15gb8c.proxy.runpod.net/query \
  -H "Content-Type: application/json" \
  -H "X-API-Key: $API_KEY" \
  -d '{"basket_query":"blue trucker hat\nblack snapback\nsummer sale"}'
payload = { "basket_query": "blue trucker hat\\nblack snapback\\nsummer sale" }
resp = requests.post(
    "https://jgypz1bf15gb8c.proxy.runpod.net/query",
    headers={"X-API-Key": api_key},

print(resp.json()["ids"])

3 · Send arbitrary JSON (POST /send)

Use /send to store any debugging or feedback payload you like; the server writes it to persistent storage. Important: the body is passed through verbatim, so structure it exactly as you want it recorded.

Headers: same as /query (plus JSON content-type).

Example body

{
  "event": "customer_feedback",
  "id": "P-20034",
  "rating": 4,
  "comment": "Pretty good, but would like more colours."
}

Response

{
  "status": "stored"
}
curl -X POST https://jgypz1bf15gb8c.proxy.runpod.net/send \
  -H "Content-Type: application/json" \
  -H "X-API-Key: $API_KEY" \
  -d @feedback.json
resp = requests.post(
    "https://jgypz1bf15gb8c.proxy.runpod.net/send",
    headers={"X-API-Key": api_key},
    json={"event": "customer_feedback", "rating": 4}
)
print(resp.json())
await fetch("https://jgypz1bf15gb8c.proxy.runpod.net/send", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "X-API-Key": apiKey
  },
  body: JSON.stringify({ event: "customer_feedback", rating: 4 })
});
